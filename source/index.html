Timestamp Microservice

A simple API that will convert Unix timecode (in seconds) to a natural date, or the other way around.

Both formats will be returned in a JSON object, as long as the request is valid.

Usage

Just pass a Unix timecode or a natural date as a param to this page's URL, and a JSON object will be returned with both the Unix timecode and natural date.


Passing a Natural Date

Natual dates must be formatted like this: January 1st 1970

You must pass the full month name, the day of this month with an ordinal, and a four-digit year.

Invalid requests will return an object with null values.


Passing a Unix Timestamp

Any number of seconds since, or before, the Unix Epoch is valid.

If the request includes characters which are not numbers, an object will null values will be returned.
